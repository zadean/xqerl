XML Query Update Test Suite Release Notes

This document contains the release notes for the XML Query Update Test Suite.

XQUTS 1.0.1 (July 27, 2010)

  • Changes have been made to reflect the resolutions of the following CR issues by the XML Query WG:
    • 7279 missing whitespace from expected ValStrict tests.
    • 7793 white space bugs in expected results
    • 8251 is id-function-declaration-017 in the right group?
    • 8706 [XQUTS] Test Suite test running guidelines incomplete
    • 9258 Static typing tests do not account for earlier static error
    • 9271 Coverage for upd:propagateNamespace
    • 9303 4 XQueryX tests are missing
    • 9312 id-rename-expr-035 should expect error XUDY0023
    • 9479 id-function-declaration-014
    • 9531 3 query names are (partially) misspelled in the catalog
  • As a result of the resolution of 8706, the catalog identifies whether or not to run certain test groups, based on whether the implementation being tested supports some of the option features. Test harnesses will need to be modified to take this into account. This change is documented in our Guidelines for Running the XML Query Update Test Suite.

XQUTS 1.0.0 (August 13, 2009)

  • This version tests all of the features in the third publication of the XQuery Update Facility Candidate Recommendation.

XQUTS 0.1.0 (July 6 , 2007)

  • This version does not have any tests for the update primitives.
